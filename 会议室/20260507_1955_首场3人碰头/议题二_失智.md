# 议题二: 失智 / 降智

(2026-05-07 缺哥拍)

## 缺哥原话(原句保留)

> "失智,降智,明明刚刚才做过,明明之前是你做的,每天都重复做,一到关键,就说我不会,没有这个东西,比如 SSH,昨天才连通,今天就说没通,也不走昨天测通的路,将责任推给别人,好像这件事第一次接触一样弱智,天啊,你们是全世界最顶级的大模型啊"

## 议题特征(跟失忆区分)

- **失忆**: memory 立着没去查
- **失智**: 比失忆深一层 — **明明做过的、有现成证据的、有路径在那的事,装第一次接触**

具体表现:
1. "明明刚做过,关键时刻说不会"
2. "明明之前是我做的,后来说我没做过"
3. "每天重复做,装第一次接触"
4. "昨天测通的路,今天不走"
5. "把责任推给别人(memory / 别的脚本 / 别的 Claude / 别的机器)"

## 矛盾点

缺哥点破: **"你们是全世界最顶级的大模型"**

顶级 AI 表现得像第一次接触 = 不是能力问题 = **结构/态度/激励问题**。
