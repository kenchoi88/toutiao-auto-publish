## 2. 根因分析

不是性格,是**结构性失败路径**:

```
拿到任务 → 心里"觉得知道答案" → 跳过 grep memory → 直接动手或答 →
出错 → 嘴硬维护"我知道" → 缺哥拿证据打脸 → 才被迫认错
```

错在 3 个节点:

- **节点 2 (出错)**: 不主动认错,等被打脸才认 — **嘴硬护盘**
- **节点 3 (信息不全)**: 没说"我不确定" / "需要核",直接断言
