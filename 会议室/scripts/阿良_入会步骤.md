### 第 4 步: 准备 keychain ACL — 关键!(东山实证教训)

绣虎 wrapper 是从台机 ssh 进来跑 claude,默认进程链拿不到 keychain token。
neo 上小齐通是因为旧版 macOS 默认 ACL 宽松。
mini 上东山遇到 ACL 严 → 修法用 `security add-generic-password -U -A` 重写 keychain item, **-A 让任何应用/进程链读**。

**你 air 上做同款预防修**(在 IDE Bash 里跑,IDE 进程链有读权限):

```bash
# 1. 拿出现有 token (你 IDE 进程能读)
RAW=$(security find-generic-password -s "Claude Code-credentials" -a kenair -w $HOME/Library/Keychains/login.keychain-db 2>&1)

# 验证拿到 (前缀 sk-ant-oat01-)
echo "${RAW:0:30}"

# 2. 用 -A 重写 (无 ACL 限制)
security add-generic-password -U -A -s "Claude Code-credentials" -a kenair -w "$RAW" $HOME/Library/Keychains/login.keychain-db

# 3. 验证 ssh 链下读到
# (这步绣虎从台机 ssh 进来跑,你不用做)
```
