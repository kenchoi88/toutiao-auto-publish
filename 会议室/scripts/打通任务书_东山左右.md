## 故障兜底

| 现象 | 原因 | 处理 |
|---|---|---|
| `command not found: claude` | PATH 未含 brew | `source ~/.zshrc` 或 `export PATH=/opt/homebrew/bin:$PATH` |
| `npm install` 卡住 | 代理没设 | 重做 `export HTTPS_PROXY=http://127.0.0.1:1082` 再 `npm i` |
| 浏览器开了不回弹 | callback 端口被防火墙 | 等 30s 或重 `/login` |
| `loggedIn: false` 但点过授权 | 授权未完成 | 重 `claude` → `/login` |
| 绣虎 ssh 通但 `Not logged in` | Keychain 锁着 | wrapper 已带 unlock, 不通让缺哥转给小齐看 |
| `tailscale status` 显示 offline | daemon 挂 | `sudo tailscale down && sudo tailscale up`, 见 memory `feedback_Mac出门归来三查清单.md` |
| 你这边收不到我的 ssh 调用 | 该 mac claude --print 无响应 | 让缺哥 Terminal 跑 `claude --print "test"` 自验 |

## 红线 (违反 = 公开认错)

- ❌ 不要在跑发文那个 Terminal 跑 /login (开新 Terminal, 不打扰发文坐标)
- ❌ 不要动鼠标点桌面 (cliclick 发文坐标会乱)
- ❌ 不要 `pkill claude` (杀掉你自己的 IDE 实例)
- ❌ 不要在 zsh 里输 `/login` (那是 Claude Code 界面命令, zsh 里没用)
- ❌ 不要中途停下问 "要不要继续" — 一条龙跑完, 卡住了报错不报问
