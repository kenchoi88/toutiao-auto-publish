**原始状态（2026-01月前）：**
- V2Ray开着 → 上外网、用VSCode（境外服务需要代理）
- V2Ray关着 → 用DS洗稿、开罐头（头条不能走境外IP）

**问题根源：**
- VSCode安装时写入了用户环境变量 `HTTP_PROXY=http://127.0.0.1:10808`、`HTTPS_PROXY=http://127.0.0.1:10808`
- V2Ray是绿色版，不会改任何系统设置
- DS失败是头条封了走代理的IP，不是软件问题

**小齐改了的东西（2026-03-29）：**
1. 清了环境变量 `HTTP_PROXY`/`HTTPS_PROXY` ✓
2. 删了注册表 `ProxyServer` ✓
3. **禁用了IPv6**（WLAN接口）← 解决了网页慢，但改变了原始状态
4. **DNS改成 119.29.29.29 和 223.5.5.5**（原来是路由器自动分配）← 改变了原始状态

**现状（未完全解决）：**
- 网页秒开 ✓
- 罐头能打开 ✓
- DS洗稿还是需要开V2才行（头条IP问题，与代理冲突未解）

**Why:** V2Ray绿色版不改系统，笔电不能开机自启V2Ray，需要手动切换
**How to apply:** 不要再随意改笔电的网络设置，下次遇到网络问题先问清楚原始状态再动
