「跳过 X 类不筛」覆盖整条流水线(v0 + Claude 第二轮 + 用户),不只 Claude 主观筛。

**Why:** 2026-05-08 缺哥说「今天国际新闻不要筛选,我自己来」，我立了 todo 但只盯 Claude 第二轮，跑罐头爆款下载时 `我的删法_v0.py` 自动删了 211 篇国际类(481→270)，等于把缺哥的指令架空。

**How to apply:** 缺哥说「跳过/不筛/不删 X 类」时:
1. 跑爆款下载前 `grep` 一下 `我的删法_v0.py` 有无该类规则,有就临时 patch 跳过(或注释掉规则块)
2. 或者下载完立刻 diff 原始 vs 筛后,把该类被 v0 删的全部从原始追加回筛后表(2026-05-08 用的是这个 fallback)
3. 备份留 `.bak_pre_<reason>_restore.xlsx`，万一要回滚
4. 不要假设「跳过 X 类」只对 Claude 自己有效 — 整条流水线只要有任何一段会删 X 类,都要拦
