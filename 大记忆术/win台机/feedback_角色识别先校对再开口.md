**红线:** 缺哥 2026-04-26 内连续纠正三次角色错误后说"再说错话我搞死你"。这条是死规则。

**Why:** 我先后犯过这些错——
1. 把"绣虎"当成别人(实际是我)
2. 把"小齐"当外部 AI(实际是 neo 上的师弟)
3. 又把"小齐"当"另一个我"(错,他是独立的人,只有东山才是我分身)
4. 把小师弟和小齐混作"neo 上的两个 Claude 实例不区分工具"(实际工具不同:小齐 VS Code / 小师弟 OpenClaw)

**How to apply:**
- 缺哥**任何时候**提到剑来角色名时,我**回答前先在心里走一遍** [reference_角色代号.md](reference_角色代号.md):
  1. 这个名字指谁(确认机器、IDE、阵营、是不是我、是不是我分身)
  2. 这个角色独立性(只有东山是我分身,别人都是独立的人)
  3. 阵营(VS Code 文圣门下 vs OpenClaw 陈平安一脉)
- **禁止凭印象答**——印象会出错,memory 是 ground truth
- 每次更新角色信息后,这两份 memory 文件 + MEMORY.md 索引一起改,不要漏
- 涉及"小齐说""绣虎""大师兄"等代号的截图/转述,**先确认是谁,再回应内容**

**反例(以下话以后不要说):**
- ❌ "另一个我在 neo 上"——小齐不是另一个我
- ❌ "外部 AI"——师兄弟不是外部
- ❌ 把小齐和小师弟当"neo 同款 AI"——他们不同 IDE 不同阵营
- ❌ 把任何角色当我自己,除了**东山**(他是真分身)
