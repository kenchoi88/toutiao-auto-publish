## 国内拳头(类比电商:猪八戒=淘宝,程序员客栈=京东)

| 平台 | 定位 | 适合度 |
|------|------|--------|
| **程序员客栈** | 程序员垂直,质量最高,B 端为主,3-10w 单价区间 | ⭐⭐⭐⭐⭐ 主战场 |
| **猪八戒网** | 国内最大威客,流量第一,综合品类,但卷+抽成高+价格被压 | ⭐⭐ 试水/铺量 |
| **一品威客** | B 端口碑好,客户预算高于猪八戒,审核严(洁具那种 B2B 单在这类) | ⭐⭐⭐⭐ 备战 B 端 |
| **猿急送** | 互联网/电商/汽车多行业,几万-十几万,纯 B 端 | ⭐⭐⭐⭐ |
| **码市(Coding)** | 程序员客栈同款,腾讯背景 | ⭐⭐⭐⭐ |
| **电鸭社区** | 远程岗,氛围好,骗子少,长期单多 | ⭐⭐⭐⭐ |

## 缺哥英文边界(关键)

**Claude 能 cover 90%**(异步文字):Profile / Cover Letter / 聊天 / 邮件 / 合同 / 报价 / 代码注释 / 文档,英文母语级输出无压力。

**Claude cover 不了的 3 个真痛点**:
1. **Zoom 视频面试** — 我没法实时上线代说,$5k+ 高单价客户 30% 会要;绕开方案:Profile 写 "prefer async via text",或用腾讯会议 AI 同传 / Otter.ai 字幕兜底
2. **时差响应** — 美东客户凌晨发消息,24h 不回可能丢单;Upwork APP 强提醒 + 常见问题英文模板备好,响应压到 5min
3. **新号 0 评价冷启动** — 头 3-5 单可能要低价吃单刷评价(国内 5k 的活 Upwork 头单 $200-500),所有人都要交的"新手税",1-2 个月跨过

## 学习接单 4 步(零风险路径)

1. **注册不投单**——程客 + 猪八戒先开号,**逛 1-2 周需求广场**,只看不投
2. **挑 1-2 单完全在能力范围内的小单试投**——500-2000 元区间练 proposal/沟通/报价
3. **第一单不计成本走完闭环**——接单→沟通→交付→收款→评价,哪怕亏本
4. **拿 1-2 个 5 星评价后再正式开始**——平台冷启动主要看评价数

## 实战路径(按时序)

1. 国内拳头双家跑顺(1-2 个月练手 + 攒作品)
2. 同时开 Upwork 慢热(profile 英化挂上,有空投 5-10 单)
3. 国内稳定后切重心到 Upwork(美元单价 2-3 倍)
4. 视频会议一律避开 / AI 同传兜底,只接 fixed-price + 异步合作类客户
