1. 源机 random 选 150 篇 docx → tar 到 `/tmp/wbXXX.tar.gz`
2. scp 源 → Win 中转 → 目标(走 Tailscale 100.x)
3. 目标机 cd 到 `~/Desktop/微头条自动发布/素材/` → tar xzf 解压
4. **ssh 源机 rm 那 150 个 docx** ← 删源关键步,scp 不会自删
5. 数学校验:`comm -12 mini_list air_list | wc -l == 0`;源机数 = 迁移前 - 150 - 期间发掉数
6. /tmp 三处(源/中转 Win/目标)残留清零
